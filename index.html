About This Project

This is a demonstration of GitHub MCP integration with Cursor. The project showcases how to create repositories, branches, and pull requests using GitHub's API.

This enhanced version includes responsive design and modern styling to improve user experience.

Key Features

Repository Management

Create and manage GitHub repositories directly from Cursor.

Branch Operations

Create branches, switch between them, and manage code changes efficiently.

Pull Request Workflow

Create pull requests, add comments, and manage the review process.
